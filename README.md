# Customer segmentation using clustering algorithms (Machine Learning)

## Purpose
- Analyze data from clients and segment them through clusterization models, 
based on sex, age, salary (dollars per year) and shopping score.

## Project link (Published on Streamlit):
- https://share.streamlit.io/rafaelcoelho1409/customersegmentation/customer_segmentation.py

## Resources
- Visual Studio Code
- python3.8
- virtualenv
- pip3: python3.x packages manager

## Python packages
- streamlit
- pandas
- plotly
- numpy
- sklearn (scikit-learn) 

## Clustering algorithms
- Principal Component Analysis (PCA)
- PCA + KMeans
- PCA + Agglomerative Clustering

## Used dataset to develop this project:
- https://www.kaggle.com/harshsoni7254/amazon-customer-segmentation

## Running this repo in your local machine:
- clone this repo:  
> git clone https://github.com/rafaelcoelho1409/CustomerSegmentation.git
- install required packages in the 'requirements.txt' file:
> pip3 install -r requirements.txt
- choose your Python interpreter (python3.x)  
- run the following commands (for Linux):
> cd CustomerSegmentation  
> streamlit run customer_segmentation.py  
- With these commands, the web page will be opened automatically. In the case it doesn't open, go to the browser and type:  
> http://localhost:8501

## Comments about this project
- In the part of the page which some clustering algorithm is applied, according to the increasing/decreasing of the number of clusters, automatically the page shows the data associated with each cluster and the associated graphs with.
